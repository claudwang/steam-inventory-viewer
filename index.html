解决浏览器 CORS 限制

Steam 对跨域请求做了严格限制，导致网页无法直接读取库存数据。以下几种方式可以解决：

⭐ 方式一：安装 CORS 浏览器扩展（推荐）

安装以下任意一个扩展，然后刷新页面重试：

Chrome: Allow CORS Firefox: CORS Everywhere

⚠️ 注意：使用时只对此页面启用，其他网站使用完后及时关闭

方式二：本地运行（更安全）

下载源码后用本地服务器运行，无 CORS 限制：

python -m http.server 8080

然后访问 http://localhost:8080

方式三：自建 Cloudflare Worker（永久方案）

在 Cloudflare Workers 部署一个专用代理，详见项目 README。
